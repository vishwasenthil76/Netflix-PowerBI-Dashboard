# 🎬 Netflix Movies & TV Shows Dashboard (Power BI)

## 📌 Overview

This project is an interactive Power BI dashboard built using the Netflix Movies & TV Shows dataset.

The dashboard provides insights into:

- Total Titles
- Movies vs TV Shows
- Top Genres
- Top Countries
- Content Rating Distribution
- Content Added by Year
- Monthly Content Added

---

## 🛠 Tools Used

- Power BI
- Power Query
- DAX
- Data Cleaning
- Data Visualization

---


## 📂 Dataset

Netflix Movies & TV Shows Dataset

---

## 🚀 Features

- Interactive Dashboard
- KPI Cards
- Donut Chart
- Line Chart
- Bar Charts
- Dynamic Filtering
- Data Cleaning using Power Query
